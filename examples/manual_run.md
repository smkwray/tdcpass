# Manual run example

## Demo run

```bash
tdcpass demo
```

Outputs:

- `data/examples/synthetic_quarterly_panel.csv`
- `output/demo/accounting_summary.csv`
- `output/demo/lp_irf.csv`
- `site/data/overview.json`

## Discover sibling caches

```bash
tdcpass discover-cache
```

tdcpass fetch-fred WTREGEN --out data/raw/fred/WTREGEN.csv
