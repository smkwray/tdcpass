# Data directories

- `raw/` — direct downloads or copied artifacts
- `cache/` — temporary or shared caches
- `derived/` — reproducible pipeline outputs
- `examples/` — synthetic example data used locally by tests and demos
- `../site/data/` — mirrored site-ready exported artifacts

Core rule:
raw and cache data can be recreated; derived data should be generated by code; examples may exist locally for smoke tests and demos but are currently ignored by the repo.
